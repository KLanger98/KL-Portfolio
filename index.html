About Me.

Hello, My name is Karl Langer.

Through the start of 2024 I will be looking to complete the Adelaide University Coding Bootcamp and kickstart my career as a Fullstack Web Developer. Using resources such as Harvard's CS50, The Odin Project and FreeCodeCamp I have been self-teaching myself to code over the last year and a half. It is only recently that I have decided to take the steps to make software development a career.

Outside of coding, I find joy in traveling, photography, bouldering, and hiking. I have spent the last three years of my professional career as a Physiotherapist and I am looking to make the move into software development. I am particularly interested in furthering my understanding of the intersection of software and healthcare.

My current skill set includes:

  • Javascript
  • jQuery
  • SQL
  • React
  • Bootstrap
  • Git
